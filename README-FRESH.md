# 🎡 Kayaverse Wheel - Phase 1

Professional spinning wheel game for the Kayaverse ecosystem. This is a Telegram Web App that integrates with the Doll bot for budget management, daily limits, and user account tracking.

## 🌟 Features

- **Beautiful Thirdweb-style UI** - Dark theme with floating orbs and professional animations
- **Smooth 6-second spin animations** - Ease-out cubic bezier curves
- **8-segment color wheel** - Premium, rainbow gradient segments
- **Real-time results display** - Glowing text, emoji animations, balance updates
- **Daily budget enforcement** - $3.00 max payouts per day
- **User daily limits** - 1 spin per user per day
- **Responsive design** - Works on all devices and screen sizes
- **Telegram Web App optimized** - Safe area support, mobile-first design
- **Kaya D. Dawg personality** - Randomized personality messages
- **Mobile game promotion** - Tournament hype messaging
